# Rahma — Islamic Companion App

Offline-first Islamic companion. No ads, no tracking, no account, no monthly cost.

**[▶ Open the web demo](./app/)** — runs the v1.0.0 build in your browser. Some features (compass, lock-screen audio controls, prayer notifications) are Android-only and will be inactive on web; everything else works.
